 # LeNet-5 论文阅读笔记
 
 > **论文**: Yann LeCun, Léon Bottou, Yoshua Bengio, Patrick Haffner.
 > *Gradient-Based Learning Applied to Document Recognition*, Proceedings of the IEEE, 1998.
 
 ---
 
 ## 1. 背景与动机
 
 - 1990 年代，手写识别主要依赖手工设计的特征 + 分类器（如 SVM、KNN）。
 - LeCun 等人提出用**卷积神经网络（CNN）** 端到端学习特征，替代手工特征工程。
 - LeNet-5 是这一思想的经典实现，专为银行支票/邮政编码上的手写数字识别设计。
 
 ---
 
 ## 2. LeNet-5 模型结构
 
 ```
 输入: 32×32 灰度图
   ↓
 C1: Conv(6@28×28)   — kernel 5×5, stride 1, 无 padding → 6 个特征图 28×28
   ↓
 S2: AvgPool(6@14×14) — 2×2, stride 2, 下采样
   ↓
 C3: Conv(16@10×10)  — kernel 5×5, stride 1 → 16 个特征图 10×10
   ↓
 S4: AvgPool(16@5×5)  — 2×2, stride 2
   ↓
 C5: Conv(120@1×1)   — kernel 5×5 → 120 个特征图 1×1（等价于 FC）
   ↓
 F6: FC(84)           — 全连接层, 84 个神经元
   ↓
 OUTPUT: FC(10)       — 10 类（数字 0–9），原始论文用 RBF，现代用 Softmax
 ```
 
 **关键设计要点**:
 
 1. **卷积层局部连接 + 权值共享** — 每个神经元只连接输入的一个局部区域，同一特征图共享同一组权重，大幅减少参数量。
 2. **交替下采样** — 池化层（平均池化）逐步降低特征图分辨率，增加感受野。
 3. **稀疏连接表（C3 层）** — C3 的 16 个特征图不是与 S2 的所有 6 个特征图全连，而是按照预设的连接表部分连接，进一步减少参数量。现代实现通常省略此优化。
 4. **输出层用 RBF（径向基函数）** — 每个输出神经元计算输入向量与该数字类原型向量的欧氏距离。现代实现常用 LogSoftmax + NLLLoss 或 CrossEntropyLoss 替代。

 ## 3. 训练细节（原始论文）
 
 | 超参数 | 值 |
 |---|---|
 | 优化器 | SGD（随机梯度下降） |
 | 学习率 | 随时间衰减（约 0.0005 量级） |
 | 权重初始化 | 均匀分布随机初始化 |
 | 损失函数 | MSE（均方误差，配合 RBF 输出） |
 | 激活函数 | Tanh / Sigmoid |
 | 训练轮数 | 约 20 epoch |
 | 批量大小 | 未明确指定（约几十到几百） |
 | 数据预处理 | 归一化到[-1, 1]或[0, 1]，仿射变换做数据增强（可选） |
 
 ---
 
 ## 4. 实验结果
 
 - MNIST 测试集错误率: **0.95%**（LeNet-5, 无数据增强）
 - 使用数据增强 + 更大模型: 约 0.8%
 - 对比方法: 当时的 KNN 约 3–5% 错误率，LeNet-5 显著领先
 
 ---
 
 ## 5. 历史意义
 
 - 确立了现代 CNN 的基本架构范式：Conv → Pool → Conv → Pool → FC → FC
 - 证明了端到端学习的有效性
 - 为后续 AlexNet（2012）、VGG（2014）等深度 CNN 奠定基础
 - 文档分析领域的奠基之作，影响 NLP、OCR、支票识别等多个方向

 ## 6. 本项目对原始论文的简化/调整
 
 | 原始论文 | 本项目（PyTorch 复现） |
 |---|---|
 | 输入 32×32 | 输入 28×28（MNIST 原始尺寸），调整 padding |
 | RBF 输出 + MSE | CrossEntropyLoss + Softmax |
 | Tanh 激活 | ReLU（更易训练、避免饱和） |
 | 平均池化（AvgPool） | 最大池化（MaxPool，现代标准做法） |
 | 稀疏连接表 | 全连接映射（简化） |
 | 20 epoch | 5–10 epoch（目标 ≥ 98%） |
 | SGD + 手动调参 | Adam / SGD + Momentum |
